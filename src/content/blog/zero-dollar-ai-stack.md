# The $0 AI Stack That Powers My Business

I run a business that makes $5,000+/month.

My infrastructure costs?

**$0.**

Here's the complete stack.

## Frontend

### Astro (Static Site Generator)
**Cost:** $0
**Why:** Blazing fast, SEO-friendly, zero JavaScript by default
**Use case:** This blog

### GitHub Pages (Hosting)
**Cost:** $0
**Why:** Unlimited bandwidth, global CDN, automatic deploys
**Use case:** Hosting astro sites

### Cloudflare (CDN + DNS)
**Cost:** $0
**Why:** Faster load times, DDoS protection, analytics
**Use case:** Custom domains + caching

## Backend & Database

### Supabase (PostgreSQL)
**Cost:** $0 (500MB database)
**Why:** Real-time subscriptions, Auth, Row Level Security
**Use case:** User data, analytics, content storage

### n8n (Automation)
**Cost:** $0 (self-hosted)
**Why:** Unlimited workflows, no execution limits
**Use case:** Content pipeline, data syncing, notifications

## AI & ML

### OpenRouter
**Cost:** Pay-per-use (~$20/month for my volume)
**Why:** One API for GPT-4, Claude, Gemini
**Use case:** Content generation, summarization

### Gemini Pro (Free Tier)
**Cost:** $0 (60 requests/minute)
**Why:** Google's best model, generous free tier
**Use case:** Draft editing, research

### Replicate
**Cost:** Pay-per-use (~$5/month)
**Why:** Run open-source AI models
**Use case:** Image generation, transcription

## Marketing & Communication

**Cost:** $0 (up to 1,000 subscribers)

**Use case:** Email newsletter

### Telegram Bot API
**Cost:** $0
**Why:** Instant messaging, automation
**Use case:** Notifications, alerts

### Notion (CMS)
**Cost:** $0 (personal use)
